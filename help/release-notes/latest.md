# Current Customer Journey Analytics release notes

| Funzione | Descrizione | Data definita |
| ----------- | ---------- | ----- |
| [!UICONTROL Persistence] | Quando crei o modifichi una visualizzazione dati, puoi associare la persistenza di una dimensione a un’altra dimensione o metrica. Questo concetto è noto come _merchandising_ in Reports &amp; Analytics ed è ora supportato in CJA. [Ulteriori informazioni](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-dataviews/component-settings/persistence.html?lang=it#binding-dimension) | 19 gennaio 2022 |
| [!UICONTROL First Known][!UICONTROL Last Known] | Questi due nuovi modelli di allocazione considerano il primo o l’ultimo valore osservato come dimensione all’interno di un ambito di persistenza specificato (sessione, persona o periodo di tempo personalizzato con look-back). Quindi applicano il modello di allocazione a tutti gli eventi all’interno dell’ambito specificato. [Ulteriori informazioni](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-dataviews/component-settings/persistence.html?lang=it#allocation-settings) | 19 gennaio 2022 |
| [!UICONTROL PersonID][!UICONTROL PersonID namespace] | Espone `personID` (o `customerID`, o qualsiasi ID utilizzato per unire i set di dati in una connessione) come dimensione nelle visualizzazioni dati. Questo miglioramento consente di includere più facilmente `personID` come dimensione nella visualizzazione dati estraendola dalla connessione. [Ulteriori informazioni](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-dataviews/component-reference.html?lang=it#optional-standard-components) | 19 gennaio 2022 |
